The website offers a highly interactive and gamified user experience with the following core features:

Dynamic Theme System:

Light Mode (Sun Horizon): The default view featuring a bright, clean aesthetic with a "Sun Horizon" background effect.

Dark Mode (Souls-like): A "bonfire" inspired dark theme with gold accents and a deep background, triggered by the toggle button.

Interactive Navigation:

Glassmorphism UI: Navigation menus, content cards, and forms utilize a frosted glass effect for a modern look.

Custom Cursor: A specialized cursor with a trailing animation that changes style based on the active theme.

Scroll-to-Top: A floating arrow button appears after scrolling down to quickly return to the hero section.

Visual Showcases:

Gallery Carousel: An interactive image slider allowing users to browse personal photos.

Certificate Grid: A responsive layout displaying professional certifications from MathWorks and Wadhwani Foundation.

Contact System: A functional contact form integrated with EmailJS for direct messaging.
